# taskman

A small task manager utility.

Currently supporting Linux. More platforms coming soon.

## Prerequisites

On Ubuntu and derivaties e.g. Mint:

```sh
sudo apt install libprocps-dev
```

## Build

```sh
cd $PROJECT_ROOT
mkdir build
cd build
cmake ..
```

If on Linux, proceed:

```sh
make
```
